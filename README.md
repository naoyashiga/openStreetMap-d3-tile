Dummy geo data is around Tokyo Tower on open street map :tokyo_tower:


##Dummy Data

```js
// Tokyo Tower Geo Data
const location = {
  latitude: 35.658581,
  longitude: 139.745433
}
```

#Usage

##Start
```
$npm start
```

##Build
```
$npm run production
```
